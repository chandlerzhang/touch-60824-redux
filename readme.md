# 初始化命令

## 生成 package.json

    npm init

## 安装 npm 插件

    npm install --save
    npm install --save-dev


## 创建 .babelrc

    {
      "presets": ["react", "es2015", "stage-2"]
    }


# 全局插件

    npm install -g babel webpack webpack-dev-server

# decorator

    npm install babel-plugin-transform-decorators-legacy --save-dev
